Array

Datastructuur die bestaat uit een lijst van elementen. Ieder element heeft een unieke index waarmee dat element aangeduid kan worden.

Cache

Opslagplaats waarin gegevens tijdelijk worden opgeslagen om sneller toegang tot deze data mogelijk te maken.

Cloud Firestore

NoSQL-documentdatabase waarmee gegevens voor mobiele en web-apps opgeslagen, gesynchroniseerd en opgevraagd kunnen worden - op wereldwijde schaal.

Firebase

Door Google ontwikkeld platform voor het maken van mobiele en webapplicaties.

GitHub repository

Hoofdmap van een project. Hierin staan alle bestanden van je project en de historie van wijzigingen die je hebt gedaan.

NFC

Near-field communication of NFC is een contactloze communicatiemethode op korte afstand, vaak tussen enerzijds een apparaat met een voeding en anderzijds een kaart of ander object met een chip en een antenne, die gezamenlijk een NFC-tag genoemd worden.

ReadMe

Computerbestand dat vaak bij een softwarepakket wordt geleverd, en dat belangrijke algemene informatie bevat waarvan de ontwikkelaars of verspreiders menen dat de gebruiker die eerst moet lezen, voorafgaand aan compilatie, installatie of eerste gebruik.

RFID

Radio-frequency identification of RFID is een techniek waarmee data opgeslagen wordt op een minuscule chip. Via radiogolven leest een elektromagnetische lezer de gegevens op de chip af.

SQL

Een SQL-database is een relationele database die opgemaakt is uit tabellen met gestructureerde gegevens, en die compatibel is met de Structured Query Language (SQL). Met behulp van SQL kan uit de database informatie opgevraagd worden en kunnen gegevens worden aangepast.

UI

User interface​, ​de middelen waarmee de gebruiker en een computersysteem met elkaar interacteren.

Use case

Beschrijving van een gedrag van een systeem, dat reageert op een verzoek dat stamt van buiten het systeem.
